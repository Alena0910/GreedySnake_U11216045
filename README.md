# 貪食蛇
## 基本版規則
 - 蛇不能穿牆，超過邊界遊戲結束。
 - 蛇碰到自己遊戲結束。
 - 蛇吃到水果得分，身體長度 + 1。

## 愚人節版規則
 - 蛇不能穿牆，超過邊界遊戲結束。
 - 蛇碰到自己遊戲結束。
 - 遊戲勝利條件：分數達到 10 分。

### 模式
吃到水果可能出現的狀況。

1. 蛇的身體 及 分數 + 2。
2. 蛇速度 * 2。
3. 蛇的身體 及 分數 - 1。
4. 清空分數 及 蛇的身體。
5. 水果移動 ( 一秒移動一次 )，吃到分數 + 1。

## 排行榜
 - 記錄基本版前十最高分數
 - 記錄愚人節版前十結束最高分數 ( 零分或負分不記錄 )
 - 記錄愚人節版前十當次遊玩產生過的最高分數
